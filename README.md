# Hookah Lounge Prototype

Быстрый прототип игры про управление кальянной с фокусом на мобильный релиз в будущем.

## Что уже есть

- Цикл игрового дня (180 секунд)
- Очередь клиентов с терпением и требованиями к качеству
- 3 зоны взаимодействия:
  - `Станция` - подготовка кальяна (тратит деньги, создает ресурс)
  - `Зона гостей` - обслуживание заказа
  - `Касса` - покупка апгрейдов
- Экономика: деньги, репутация, прокачка
- Поддержка клавиатуры и touch-кнопок

## Запуск

Можно открыть `index.html` напрямую в браузере.

Для корректного mobile-пайплайна лучше через локальный сервер:

```bash
npx serve .
```

## Управление

- `WASD` / стрелки - движение
- `E` - действие
- На телефоне: кнопки под экраном

## Как довести до релиза на телефон

1. Перенести проект на сборку 
2. Добавить Capacitor:
   - `npm i @capacitor/core @capacitor/cli`
   - `npx cap init`
   - `npx cap add android` (и `ios` на macOS)
3. Настроить ассеты, splash, иконки, уведомления.
4. Добавить сохранение прогресса (localStorage/облако).
5. Добавить контент: больше гостей, предметов, долгосрочная мета-прогрессия.
